Why have we done all of this? Simple: to count the number of ways to distribute 7 cookies to 4 kids, all we need to do is count how many sticks and stones charts there are. But a sticks and stones chart is just a string of symbols, some sticks and some stones. If instead of sticks and stones we would use 0’s and 1’s, it would just be a bit string. We know how to count those.
