A multiset is a collection of objects, just like a set, but can contain an object more than once (the order of the elements still doesnâ€™t matter). For example, \(\{1,1, 2, 5, 5, 7\}\) is a multiset of size 6.
  1. How many sets of size 8 can be made using the 10 numeric digits 0 through 9?
  2. How many multisets of size 8 can be made using the 10 numeric digits 0 through 9?
Solution.
  1. \({10\choose 8}\) sets. We must select 8 of the 10 digits to put in the set.
  2. Use stars and bars: each star represents one of the 8 elements of the set, each bar represents a switch between digits. So there are 8 stars and 9 bars, giving us \({17 \choose 8}\) sets.
